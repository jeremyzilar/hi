# Over The Horizon

A conference/gathering project planning document for a potential event at Ghost Ranch, New Mexico.

## Overview

This project documents the planning and conceptualization of a small, intimate gathering inspired by the High Ground conference model. The goal is to create a space for meaningful conversation and connection around design, culture, and what's happening in the world.

## Project Goals

- Determine feasibility and costs of hosting at Ghost Ranch
- Define the optimal group size and event duration
- Structure the gathering format
- Articulate the purpose and values of the event

## Documentation Structure

- **[Big Questions](./big-questions.md)** - Key questions to answer in the planning process
- **[High Ground Model](./high-ground-model.md)** - Details from the Santa Fe conference that inspired this
- **[Ghost Ranch Information](./ghost-ranch-info.md)** - Venue details, costs, and logistics
- **[Cost Analysis](./cost-analysis.md)** - Sweet spot calculations for group size and duration
- **[Structure & Format](./structure-format.md)** - How the gathering would be organized
- **[Purpose & Values](./purpose-values.md)** - What we want to get out of this
- **[Planning Checklist](./planning-checklist.md)** - Next steps and action items

## Quick Reference

- **Venue**: Ghost Ranch, New Mexico
- **Model**: High Ground conference format (Santa Fe)
- **Core Principle**: Friends gathering to stay clued in to what's happening in the world
